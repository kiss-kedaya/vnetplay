# UI Guidelines

## Design direction
- Reference: Vercel-quality minimal dashboards.
- Tone: premium, calm, sharp, spacious.
- Use neutral grayscale as the base and one restrained accent color.

## Rules
- No emoji in product UI, docs, or commits.
- Use SVG or icon components instead of emoji.
- Avoid visual noise, bloated gradients, and heavy decoration.
- Cards need crisp borders, subtle shadows, and clear spacing.
- Navigation should stay simple: left sidebar, top toolbar, focused content area.
