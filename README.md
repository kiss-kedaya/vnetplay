# VNetPlay

VNetPlay is a modular LAN gaming platform built with Rust + Tauri for the desktop client and Rust for the control server.

## Goals
- Fast virtual LAN experience for games such as Minecraft and LAN-based titles.
- Clean Vercel-inspired desktop UI.
- Strong modularity and maintainability.
- Self-hosted control plane and relay-friendly topology.

## Repository layout
- `app/` frontend application
- `desktop/` Tauri desktop shell and Rust core
- `server/` Rust control server
- `docs/` plans, architecture, and UI rules
